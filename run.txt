# Setup python, venv
1. Installing venv on windows
download pip.
download venv
run python -m venv (enviorment_name)
./(enviorment_name)/Scripts/activate
2. Installing virtualenv on MAX / Linux
  a. pip install virtualenv
  b. cd into base directory of project 
  c. virtualenv venv
  d. source venv/bin/activate
